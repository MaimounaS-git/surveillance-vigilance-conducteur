# Surveillance de la vigilance du conducteur

Un programme qui utilise une webcam pour repérer les signes de fatigue au volant (yeux qui se ferment, bâillements) et prévenir le conducteur avant qu'il ne s'endorme.

Projet de fin d'études (Licence 3 Big Data / Data Science).

## Lancer la démo

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python src/main.py
```
